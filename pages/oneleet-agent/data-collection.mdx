# Data Collection

The Oneleet Agent collects data from your device for continuous monitoring and security. The data collected includes:

<Tabs items={['macOS', 'Windows', 'Linux']}>
  <Tabs.Tab>
### System information

- Computer name
- CPU brand
- Hardware model
- Hardware serial

### OS information

- OS name
- OS version
- OS architecture

### Firewall status

Firewall is a network security system that monitors and controls incoming and outgoing network traffic. We check the status of the firewall to ensure that it is enabled.

### Gatekeeper status

Gatekeeper is a security feature in macOS that helps protect your Mac from malicious software. We check the status of Gatekeeper to ensure that it is enabled.

### FileVault status

FileVault is a disk encryption program in macOS that helps protect your data. We check the status of FileVault for your main disk drive to ensure that it is enabled.

### Screenlock status

Screenlock is a security feature that requires a password to unlock your device. We check the status of screenlock to ensure that it is enabled and set to a minimum threshold.
