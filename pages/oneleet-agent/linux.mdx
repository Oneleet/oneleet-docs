## Data collection

The Oneleet Agent collects data from your device for continuous monitoring and security. The data collected includes:

### System information

- Computer name
- CPU brand
- Hardware model
- Hardware serial

### OS information

- OS name
- OS version
- OS architecture

### Disk encryption status

Disk encryption is a security feature that helps protect your data. We check the status of disk encryption for your main disk drive to ensure that it is enabled.
