# Setup

<details>
<summary>Add employees to your tenant (Admins only)</summary>

1. Go to the [Oneleet Dashboard](https://app.oneleet.com).
2. Click on the `People` tab.
3. Make sure everyone you want to add is either a tenant member or an employee.
<Callout type="info">
  An employee has access to the [Oneleet Portal](https://portal.oneleet.com), while a tenant member has access to both the [Dashboard](https://app.oneleet.com) as well as the Portal.
</Callout>
</details>

  # macOS (Beta)

## System requirements

Minimum system requirements for the Oneleet Agent on macOS:

- macOS 13 (Ventura)
- 8 GB RAM

## Download

<Tabs items={['Directly from Oneleet Portal', 'Homebrew']}>
  <Tabs.Tab>
    1. Open the [Oneleet Portal](https://portal.oneleet.com) in your browser and log in.
    2. Click on the **Devices** tab to download the Oneleet Agent for macOS.

    ### Install

    The above steps should have downloaded a DMG to install the Oneleet Agent on your Mac.

    1. Double-click on the DMG to mount it
    2. Drag and drop the Oneleet app bundle into the **Applications** folder
    3. You can now search for Oneleet in **Launchpad** to launch the Agent
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    brew tap oneleet/tap
    brew install --cask oneleet-agent
    ```
  </Tabs.Tab>
</Tabs>

## Uninstall

<Callout type="warning">
  This will remove the Oneleet Agent from your device and stop all monitoring of
  your system state which can affect your compliance and security.
</Callout>

<Tabs items={['Directly from Oneleet Portal', 'Homebrew']}>
  <Tabs.Tab>
    1. Open **Finder** and go to the **Applications** folder.
    2. Locate the Oneleet app bundle.
    3. Drag the app to the **Bin** or right-click and select **Move to Bin**.
    4. Empty the Trash by right-clicking on the **Bin** icon and selecting **Empty Bin**.
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash
    brew uninstall oneleet-agent
    ```
  </Tabs.Tab>
</Tabs>

# Windows (Beta)

## System requirements

Minimum system requirements for the Oneleet Agent on Windows:

- Windows 10 x64
- 8 GB RAM

## Download

1. Open the [Oneleet Portal](https://portal.oneleet.com) in your browser and log in.
2. Click on the `Devices` tab to download the Oneleet Agent for Windows.

## Install

The above step should have downloaded a setup EXE to install the Oneleet Agent on your PC.

1. Double-click on the setup EXE to run it.
2. Follow the on-screen instructions to install the Oneleet Agent.

## Uninstall

<Callout type="warning">
  This will remove the Oneleet Agent from your device and stop all monitoring of
  your system state which can affect your compliance and security.
</Callout>

1. Open the **Control Panel** and navigate to the **Add or remove programs** section
2. Scroll to find **Oneleet**.
3. Click **More** (three dots) next to it, then select **Uninstall**.

    # Linux (Alpha)

## System requirements

Minimum system requirements for the Oneleet Agent on Linux:

- Debian or RPM-based distro
- arm64 or amd64
- 8 GB RAM

## Install

**Ubuntu, Debian, Mint**

```bash
sudo apt install curl
sudo curl -fsSLo /usr/share/keyrings/oneleet-agent-archive-keyring.gpg https://downloads.oneleet.com/oneleet-agent-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/oneleet-agent-archive-keyring.gpg] https://downloads.oneleet.com/agent/linux/debian stable main" | sudo tee /etc/apt/sources.list.d/oneleet-agent-release.list
sudo apt update
sudo apt install oneleet-agent
```

**Fedora, Rocky/RHEL**

```bash
sudo dnf install dnf-plugins-core
sudo dnf config-manager --add-repo https://downloads.oneleet.com/oneleet-agent.repo
sudo rpm --import https://downloads.oneleet.com/oneleet-agent.asc
sudo dnf install oneleet-agent
```

**OpenSUSE**

```bash
sudo zypper install curl
sudo rpm --import https://downloads.oneleet.com/oneleet-agent.asc
sudo zypper addrepo https://downloads.oneleet.com/oneleet-agent.repo
sudo zypper install oneleet-agent
```

## Uninstall

<Callout type="warning">
  This will remove the Oneleet Agent from your device and stop all monitoring of
  your system state which can affect your compliance and security.
</Callout>

**Ubuntu, Debian, Mint**

```bash
sudo apt remove oneleet-agent
```

**Fedora, Rocky/RHEL**

```bash
sudo dnf remove oneleet-agent
```

**OpenSUSE**

```bash
sudo zypper remove oneleet-agent
```
